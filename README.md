### Тестовое задание на вакансию. Разработчик в тестировании

Данное тестовое задание предназначено для выявления следующих навыков: 
* Умение работать самостоятельно; 
* Скорость выполнения задач; 
* Работа с современными технологиями. 

Во время выполнения тестового задания важно по максимуму проявить эти 
качества. Удачи! 

При выполнении работы необходимо использовать следующие 
технологии:

1) Selenium GRID (тесты запускать через GRID обязательно) 
2) Использовать паттерн проектирования автотестов Page Object 
3) Реализовать формирование отчетов о пройденных тестах через Allure 
В задании необходимо: 
4) Использовать Python/Java, подключить библиотеку Selenium Webdriver; 
5) С помощью Selenium открыть браузер, открыть страницу страницу 
https://www.globalsqa.com/angularJs-protractor/BankingProject/#/login; 
6) Авторизоваться пользователем «Harry Potter»; 
7) Вычислить N-е число Фибоначчи, где N - это текущий день месяца + 1. 
Пример: сегодня 08.02.2023, 9-е чисто Фибоначчи равно 21; 
8) Выполнить пополнение счета (Deposit) на сумму равную вычисленному в 
п.4 числу; 
9) Выполнить списание со счета (Withdrawl) на сумму равную вычисленному 
в п.4 числу; 
10) Выполнить проверку баланса - должен быть равен нулю; 
11) Открыть страницу транзакций и проверить наличие обеих транзакций; 
12) Сформировать файл формата csv, в который выгрузить данные о 
проведенных транзакциях; 
Файл должен содержать строки следующего формата 
<Дата-времяТранзакции Сумма ТипТранзакции>, где 
Формат Дата-времяТранзакции - "ДД Месяц ГГГГ ЧЧ:ММ:СС" 
Формат Сумма - число 
Формат ТипТранзакции - значение из списка [Credit, Debit] 
13) Оформить сформированный файл как вложение к отчету allure. 


Ссылку на github-проект выполненного тестового задания и информацию о 
трудозатратах указать в maketalents.

**Запуск:**
1. Запустить Hub - `java -jar selenium-server-4.20.0.jar hub`
2. Запустить Node - `java -jar selenium-server-4.20.0.jar node --detect-drivers true --publish-events tcp://172.18.224.1:4442 --subscribe-events tcp://172.18.224.1:4443 --selenium-manager true`
3. Запустить тесты - `pytest --maxfail=1 --disable-warnings`
